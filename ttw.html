Installation

Installing Tale of Two Wastelands

  1. Download the Tale of Two Wastelands Installer and extract the contents to your Downloads folder.
  2. Download the Universal MPI Phraser. Extract the contents to the already existing Tale of Two Wastelands Installer folder.
  3. Open Dolpin
  4. Navigate to the folder where you installed Mod Organizer 2.
  5. Here, create a folder named Tale of Two Wastelands.
  6. Take note of this new folder's path. If you installed MO2 in the default folder it will be
    7. ~/modorganizer2-newvegas/The Best of Times/mods/Tale of Two Wastelands
  7. Navigate to your TTW Installer folder. Double click the file named ttw_linux_gui. This will launch the Linux TTW installer.
  8. Click Browse. Select the file marked Tale of Two Wastelands 3.4.mpi
  9. Locate your installed copies of Fallout 3, Fallout: New Vegas & Mod Organizer 2. The Output path box should point to the Tale of Two Wastelands mod directory you created earlier.
  10. Ensure your paths look similar to the example above, then click Install.

The installation might take a long time as the audio encoding section is very CPU-intensive. It takes ~90 minutes on a Steam Deck LCD.

It is recommended to keep your handheld device connected to AC power and close any other applications during this time to avoid accidental shutdown and data corruption.

If you run into any errors during the installation, check the solutions listed in the FAQ.

Activating and Size Check

  1. Open MO2.
  2. If you installed TTW correctly, you should see the Tale of Two Wastelands mod in the left pane of MO2.
    • If you do not see anything or the mod name is greyed out, you did not install TTW to the correct folder. Find out where the files were installed to and move them to the folder you created in step 3 of the previous section.
      The path should look like this: [YOUR MO2 FOLDER]/mods/Tale of Two Wastelands
  3. Click the checkbox next to it in the left pane and the right pane should fill with plugins.
  4. Open a new Dolphin window and navigate to your Tale of Two Wastelands mod folder.
  5. Right-click anywhere in the folder (do not select any file inside) and open its Properties.
  6. The folder's size should be 16.6 GB, reinstall Tale of Two Wastelands if it isn't. This can be done by first clearing the contents of the current folder, then repeating the installation steps.
